# Glossary

| Term | Definition |
|------|-----------|
| **ThirdLife** | The game title — a persistent multiplayer life simulation |
| **Credits** | In-game currency |
| **District** | A zone within the city (Downtown, Residential, etc.) |
| **Plot** | A purchasable land parcel where buildings can be placed |
| **Room** | A Colyseus game room instance |
| **State** | Server-authoritative game state synced to clients |
| **ECS** | Entity-Component-System — the code architecture pattern |
| **HUD** | Heads-Up Display — persistent on-screen UI |
| **GDD** | Game Design Document |
| **LOD** | Level of Detail — mesh complexity based on distance |
