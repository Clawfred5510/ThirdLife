# Art Department

## Mission
Define and produce the visual identity of all studio projects.

## Roles
- **Concept Artist** (On-Demand) — Visual direction, style guides, concept art
- **UI Designer** (On-Demand) — HUD, menus, in-game UI design

## Pipeline
1. **Brief** — Receive visual requirements from Design
2. **Concept** — Create concept art / mockups
3. **Produce** — Create final assets per asset pipeline specs
4. **Review** — Art review with Director/Design Lead
