# QA Department

## Mission
Ensure quality, stability, and player experience standards are met.

## Roles
- **QA Tester** (On-Demand) — Testing, bug reports, regression checks

## Pipeline
1. **Receive** — Get build or feature to test
2. **Test** — Execute test plan, exploratory testing
3. **Report** — File bugs to taskboard/backlog
4. **Verify** — Confirm fixes pass regression
