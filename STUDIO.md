# GameStu — Virtual Game Studio

## Mission
Build ambitious, persistent multiplayer experiences using a web-first tech stack with an adaptive team of AI agents.

## Current Project
**ThirdLife** — A persistent open-world life simulation where players own buildings, run businesses, and build wealth in a shared city.

## Workflow
- **Hub & Spoke**: Director coordinates department leads
- **Pipeline**: Within each department: spec -> implement -> review -> integrate
- **Taskboard**: Bug fixes and polish tracked in `studio/taskboard/`

## Scaling
- Minimum: Director + Tech Lead + Design Lead (3 agents)
- Scale up per-department when project phase demands it
- On-demand agents activate when needed, go dormant when done (memory preserved)
